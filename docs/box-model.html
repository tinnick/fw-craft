ボックスモデルの特徴

  • 中身（innerHTML）とボーダーはスタイリング出来る
  • マージン（margin）は他のエレメント（タグで定義された箱）との間隔地を定義するもの
  • ボーダー（border）はエレメントの外枠
  • パディング（padding）は外枠と中身の間隔地を定義するもの
  • マージン、パディングはスタイリング出来ない
  • 背景のスタイリングは中身とパディングに影響するが、マージンには影響しない
